## Requirements
* Python

## Usage
```
termine.py [-h] [-hs] [-gs WxH] [-m N]
```

## Controls:
```
  Spacebar: open a cell
  F:        flag
  Q or ^C:  quit
```

## Optional arguments:
```
  -hs, --highscores        Print highscores
  -gs WxH, --gridsize WxH  Size of the grid
  -m N, --mines N          Number of mines
```
